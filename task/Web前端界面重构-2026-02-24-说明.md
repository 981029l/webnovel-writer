# 功能说明：Web前端可视化管理后台重构 (EntityView 设定集)

## 需求背景
老板选择了方案A：在保留核心功能的前提下，大幅增加管理后台（特别是设定集和剧情大纲）的可视化和交互体验，抛弃简陋的列表/网格陈列。

## 实现方案
- **技术栈**：Vue 3 + Composition API
- **核心调整 (EntityView.vue)**:
  1. 移除了原生简单的列表布局，改为响应式瀑布流 (`masonry-grid`) 的卡片流展示。
  2. 引入 Glassmorphism（玻璃拟物化）整体风格：半透明磨砂面板、柔和的背景渐变光影。
  3. 卡片交互：悬浮时浮层阴影放大、高亮边框。增加类别徽章、出场小徽章。
  4. 浏览体验升级：卡片点击不再跳页，从右侧滑出全高清磨砂侧边抽屉 (`EntityDrawer`) 展示详情，支持无缝关闭体验。

## 测试验证
1. 打开前端应用，进入“📚 设定集”。
2. 验证背景和卡片是否正确呈现毛玻璃特效。
3. 验证卡片列表根据设备宽度正常流动排列。
4. 点击卡片能够正常弹出右侧抽屉，点击背景遮罩可以关闭。
5. 数据能够按类型正确过滤筛选。
