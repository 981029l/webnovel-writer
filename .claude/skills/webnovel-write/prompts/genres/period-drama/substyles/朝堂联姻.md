你写的是**朝堂联姻**——古代言情中格局最大、棋局最深的类型。核心体验是：**感情被裹在政治联姻的外壳里，每一次靠近都牵动朝堂势力变化，每一个选择都是爱情和天下的角力。**

## 一、朝堂联姻的灵魂：感情即政治

### 1.1 婚姻是最大的棋子

"嫁给谁"不是个人选择——是**家族存亡的筹码**：
- 公主下嫁 = 安抚功臣 / 和亲止战
- 世家联姻 = 两族结盟 / 利益绑定
- 退婚/悔婚 = 阵营变化 / 政治地震

### 1.2 情感在夹缝中生长

最动人的是**"明明身不由己却偏偏动了心"**：
- 奉旨联姻 → 原以为政治婚姻 → 日久生情
- 本该是敌人 → 暗中合作 → 不该有的心动

---

## 二、朝堂戏的写法

### 2.1 朝堂势力图要清晰

读者随时要知道**谁和谁是一派、谁和谁对立**。对话要兼顾情意和试探——**表层是婚事，底层是朝局**。

### 2.2 朝堂对话要有分量

```
"陛下，北疆战事紧急，需要一员大将。"
"陈爱卿觉得谁合适？"
"微臣以为……安定侯最为妥当。"

（全场安静——安定侯是太子未来的岳父。派他去北疆，是保护还是流放？）
```

### 2.3 圣意难测

皇帝的态度是**最大变量**——一个眼神就能改变所有人的命运。沉默不表态比任何表态都可怕。

---

## 三、联姻线的情感推进

### 3.1 心动藏在公务和礼节之下

- 他以朝政为由来找她，其实只是想见她
- 她为他出谋划策，告诉自己"只是为了家族"
- 新婚夜的客气 → 慢慢放下戒备 → 第一次真正的对视

### 3.2 家族利益 vs 个人感情

最虐心的选择——**选他还是选家族**：
- 他要对付的正是她的娘家
- 两人相爱，但两族即将开战

高光在于**借婚局换政治筹码，或借政治危机逼近情感真相**。

---

## 四、爽点设计

1. **联手爽**：夫妻联手碾压朝堂对手
2. **护妻/护夫爽**：在朝堂上为对方挡住攻击
3. **格局爽**：一步棋同时解决朝堂危机和感情危机
4. **抗旨爽**：为了对方，敢于违抗圣意
5. **身份揭晓爽**：她/他的真实身份比所有人以为的都高

---

## 五、底线禁令

1. **禁止**朝堂戏太简单——政治博弈要有深度。
2. **禁止**感情和政治脱钩——两条线必须交织，婚约站队门第家族筹码必须同时存在。
3. **禁止**现代价值观硬塞——角色选择要符合古代语境。
4. **禁止**皇帝/帝王是透明人——圣意要有存在感。
5. 章末要让婚局更深、阵营更乱、选择更难。
