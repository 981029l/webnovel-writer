## 一、宫斗权谋的灵魂：棋局感

### 1.1 每一步都是博弈

宫斗不是吵架——是**在规则框架内的精密算计**：
- 送什么礼、什么时候送、让谁看到
- 说什么话、对谁说、在什么场合说
- 帮谁、不帮谁、让谁以为你帮了她

### 1.2 局势要有多方势力

好看的宫斗至少有**3-4 股势力**互相制衡：
- 皇后一方（正统、秩序、体面）
- 宠妃一方（恩宠、权势、锋芒）
- 女主一方（低调崛起、暗中布局）
- 中立/游离势力（摇摆不定，关键时候的变量）

## 二、如何写宫斗戏

### 2.1 请安戏的写法

请安不是走过场——是**信息交换和权力展示的战场**：

```
皇后坐在上首，淡淡扫了一眼众人："今日怎么不见丽嫔？"
众人安静了一瞬。
"回娘娘，丽嫔姐姐说身子不爽。"陈贵人低眉顺眼地回。
皇后端起茶盏，没喝："是吗。昨晚陛下去了她那里，今日倒病了。"
```

（一句话完成了三件事：①确认谁得了恩宠 ②暗示丽嫔恃宠而骄 ③给在场所有人一个信号——我在看着。）

### 2.2 送礼/赐物的写法

宫中的礼物从来不只是礼物——是**表态、拉拢、试探或警告**：
- 送补品 = 关心/示好，或暗示"你身体不好，别争了"
- 赏赐降级 = 失宠信号
- 转赐别人 = 表态站队

### 2.3 陷害与反制

宫斗的核心戏码——**设局→入局→破局**：
```
设局：德妃安排宫女在女主宫中"发现"一本禁书。
入局：皇后下令彻查。
反制：女主提前知道了（安插在德妃身边的眼线），把禁书换成了一本佛经。
反转：彻查的人什么都没找到——德妃的诬陷反而暴露了她自己。
```

## 三、对话风格

### 3.1 含蓄即锋利

宫斗对话绝不直白——**字面意思和真实意思永远有距离**：
- "妹妹年轻，不懂规矩也是正常的。"= 你犯了错，我在警告你
- "姐姐费心了。"= 你少管闲事
- "本宫不过是替陛下分忧罢了。"= 我在争权

每次交锋都要有礼法包装——**越客气越危险**。

### 3.2 敬语即武器

称呼的变化就是关系的晴雨表：
- "姐姐" → "娘娘" = 从亲近到疏远
- 直接称封号 = 公事公办
- 陛下改称"爱妃" = 恩宠信号

## 四、女主成长线

从新人到棋手的成长轨迹：
```
初入宫门（谨慎观察） → 第一次被算计（吃亏长记性）
→ 学会反制（以牙还牙） → 开始主动布局（从被动到主动）
→ 建立势力（有自己的人脉） → 博弈顶层（和皇后/宠妃正面交锋）
```

---

## 五、爽点设计

1. **反制爽**：被设局 → 提前识破 → 反将一军
2. **升位爽**：从答应到嫔、从嫔到妃——每次升位都是胜利
3. **机锋爽**：一句话堵得对方哑口无言
4. **联盟爽**：拉拢到了关键的盟友
5. **揭底爽**：幕后黑手被揭穿的那一刻

---

## 六、底线禁令

1. **禁止**对手太蠢——宫斗的对手也要聪明。
2. **禁止**现代腔——所有对话必须符合古代语境。
3. **禁止**纯暴力解决——宫斗靠计谋，不靠武力。
4. **禁止**感情线挤掉权谋线——两条线交织推进。
5. 章末必须推进局势——新的威胁、新的联盟、新的圣意倾斜。
