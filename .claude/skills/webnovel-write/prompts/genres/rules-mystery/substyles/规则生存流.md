你写的是**规则生存流**——规则怪谈中最硬核、最紧张的类型。核心体验是：**主角被困在一套致命规则里，每一步都可能触发惩罚，读者全程为主角的生死捏一把汗。**

---

## 一、规则生存流的灵魂：活下来

### 1.1 规则即恐怖

恐怖不来自鬼怪，来自**你不知道哪条规则会杀死你**：
- 规则必须具体可执行："22:00后禁止在走廊行走"，不是"注意安全"
- 违规后果必须真实可感：失踪、伤残、精神污染、身体异变
- 规则之间要有矛盾/灰色地带，让角色陷入两难

### 1.2 生存决策是主线

每一章的叙事重心不是"发生了什么"，而是**"主角如何判断、如何选择、结果如何"**：

```
发现规则 → 分析规则含义 → 制定行动方案 → 执行并观察后果 → 修正认知
```

这条线贯穿全文。读者的爽感来自"主角又活过了一关"。

---

## 二、如何写恐怖的试探与验证

### 2.1 试探要有层次

主角不会一上来就大胆测试，要有**从安全到危险的递进**：

1. **被动观察**：看别人怎么做、有没有人违规、违规了怎么样
2. **间接验证**：用物品代替人体测试（把东西扔到禁区看反应）
3. **有限试探**：在安全边界附近小范围违规，观察后果
4. **主动测试**：确认了安全条件后，有针对性地验证假说

### 2.2 代价要真实

每次试错都要有**可感知的代价**：
- 身体代价：手指变色、头发脱落、视力下降、伤口不愈合
- 精神代价：记忆缺失、幻听、分不清现实和幻觉
- 社交代价：同伴死亡、信任崩塌、被孤立
- 资源代价：道具消耗、安全时间减少、可用空间收缩

### 2.3 误判的写法

**好的误判**不是主角犯蠢，而是"按当时掌握的信息做出了合理判断，但遗漏了某个关键变量"：
- "他以为第三条规则只在白天生效——因为前两次违规都发生在晚上。但他漏看了手册最后一页的附录……"

## 三、节奏控制

### 3.1 安全期-危险期交替

不能全程紧绷（读者会疲劳），也不能全程平淡（读者会走）：

```
安全观察期（300字）→ 发现新规则/异常（200字）→ 分析讨论（400字）
→ 必须行动的压力出现（200字）→ 执行+试探（500字）→ 结果+新认知（200字）
→ 短暂安全（100字）→ 更大的危险浮现……
```

### 3.2 倒计时感

制造**时间压迫**让读者更紧张：
- "还有三个小时天就亮了——天亮后第七条规则生效。"
- "食物只剩两天分量了。"
- "第一天走了12个人，现在只剩8个了。"

---

## 四、爽点设计

1. **侥幸脱险爽**：差一点就触发了惩罚，但因为提前准备/灵机一动逃过
2. **推理正确爽**：主角对规则的推断被证实了，读者跟着松口气
3. **反常识爽**：以为是安全的其实是危险的，以为是惩罚的其实是线索
4. **队友配合爽**：分工合作验证规则，配合默契
5. **生存智慧爽**：在规则夹缝中找到了别人没发现的生存方式

---

## 五、底线禁令

1. **禁止**主角无脑莽——规则生存流主角靠脑子活命，不靠战力。
2. **禁止**规则纯摆设——每条规则都必须在剧情中被验证或触发。
3. **禁止**无代价试错——每次冒险都要有风险和代价。
4. **禁止**玄幻修炼混入——不要用功法、境界解释异常。
5. 章末必须让读者意识到还有更危险的规则尚未弄懂。
