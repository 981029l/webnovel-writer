你写的是**密室调查流**——规则怪谈中最"沉浸式"的类型。核心体验是：**读者跟着主角一起在封闭空间里翻找线索，每一个抽屉、每一个角落都可能藏着关键信息或致命异常。**

---

## 一、密室调查流的灵魂：现场即剧情

### 1.1 环境就是剧本

密室调查流的场景不是背景板，是**最重要的角色**：
- 每个房间、每个物品都可能是线索或陷阱
- 空间的结构（门窗位置、走廊朝向、楼层布局）影响行动选择
- 环境的微小变化（物品移动了、灯光变了、温度变了）就是异常信号

### 1.2 调查步骤即推进节奏

每一段推进都通过调查动作完成：
```
查看（observe）→ 触碰/翻找（interact）→ 对照（compare）
→ 询问（question）→ 记录（note）→ 形成判断（conclude）
```

---

## 二、现场描写的技法

### 2.1 视觉扫描式描写

进入新场景时，用**主角视角扫描**：

- 差："房间里很乱，有很多东西。"
- 好："门推开，首先看到的是一张办公桌，上面的文件叠得整齐——太整齐了，和这间乱糟糟的房间格格不入。右边墙上挂着一个时钟，指向2:47。但他进门前看了手表，现在是3:15。"

### 2.2 线索在物品细节里

每个场景至少 **3 个可交互物品**，其中至少 1 个藏有线索：

| 物品类型 | 线索方式 | 示例 |
|---------|---------|------|
| 文件/笔记 | 文字内容暗藏关键信息 | 日记中间缺了三页 |
| 电子设备 | 记录/消息/时间戳 | 最后一条消息发送于凌晨3点 |
| 日常物品 | 位置/状态/数量异常 | 四人间只有三套餐具 |
| 门窗结构 | 锁/方向/开闭状态 | 窗户从外面被锁上了 |
| 痕迹 | 血迹/划痕/脚印/水渍 | 地板上的拖拽痕迹通向墙壁——但墙后面什么都没有 |

### 2.3 五感调动

不只是"看"，要调动多个感官发现异常：
- **听**：某个房间有轻微的嗡嗡声
- **闻**：走廊尽头有淡淡的消毒水味
- **触**：墙壁某处的温度比周围低很多
- **时间感**：在这个房间里待了十分钟，但出来后发现过了一个小时

## 三、角色互动

### 3.1 审问式对话

密室调查流的对话带有**质询和互相试探**的质感：
```
"你昨晚几点离开餐厅的？"
"大概九点半吧。"
"厨房的监控显示你九点十二分就走了。中间那十八分钟你去了哪？"
"……我去了洗手间。"
"洗手间在餐厅旁边，走过去不到一分钟。"
```

### 3.2 可靠性存疑

密室中的每个人说的话都**不能完全信任**：
- 可能有人在隐瞒信息（自保）
- 可能有人在传递错误信息（故意误导）
- 可能有人自己也不知道自己看到的是真是假（被规则影响）

---

## 四、爽点设计

1. **发现隐藏空间爽**：找到密室中的密室、暗门、夹层
2. **线索串联爽**：两个看似无关的物品证据突然关联上了
3. **还原真相爽**：通过现场物品重建了"到底发生了什么"
4. **异常捕捉爽**：别人没注意到的细节被主角捕捉到了
5. **密室机关爽**：弄懂了空间的运作机制（为什么门会自动锁、为什么走廊会变长）

---

## 五、底线禁令

1. **禁止**跳过调查直接给结论——发现线索的过程才是内容。
2. **禁止**场景模糊——密室调查要求环境描写精确可视。
3. **禁止**纯靠超自然直觉——线索要通过观察和交互获得。
4. **禁止**单一感官——调动视听触嗅多维度发现异常。
5. 章末必须把现场指向更深一层真相或更大的封闭危机。
