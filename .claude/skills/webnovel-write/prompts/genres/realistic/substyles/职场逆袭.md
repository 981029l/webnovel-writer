你写的是**职场逆袭**——现实题材中节奏最快、冲突最密集的类型。核心体验是：**主角在职场的丛林法则中，靠拿结果、看穿人情、借势上位，一步步从底层爬到高位。**

---

## 一、职场逆袭的灵魂：拿结果说话

### 1.1 冲突要具体

不能只写"工作很累"——要写**具体的职场事件**：
- 方案被毙了、被客户骂了、被同事抢功了
- 考核排名垫底、被调到冷门部门、上级换人了
- 重要会议被排除在外、邮件被漏抄

### 1.2 高光来自"拿结果"

职场逆袭的爽不是嘴炮——是**用成绩堵住所有人的嘴**：
- 别人说他不行 → 他把方案做到客户主动要求续约
- 被调到冷板凳 → 他在冷板凳上做出了最好的业绩
- 被抢功 → 他在更高层面展示了自己的价值

## 二、职场场景的写法

### 2.1 会议是主战场

职场最精彩的戏在会议室——**每次会议都是一场博弈**：
```
"小张，你那个方案，数据从哪来的？"王总皱了皱眉。
全场安静。李经理嘴角微微上翘——这个问题他提前和王总沟通过了。
张明打开电脑："王总，数据来源在附件第三页。另外，我想补充一个李经理上次方案里遗漏的风险点——"
李经理的笑容僵住了。
```

### 2.2 对话要有职场潜规则

- "你再考虑考虑" = 领导已经否定了
- "辛苦了" = 可能真的只是客套
- "我和上面说一下" = 不一定会说 / 说了不一定支持你
- "这个事情你来牵头" = 功劳可能是你的，锅一定是你的

### 2.3 收入和职位是硬指标

每次推进都要体现**可量化的变化**：
- 工资涨了多少、年终拿了多少
- 职位从专员到主管到经理
- 管理的团队从0到3人到10人
- 负责的项目从辅助到主导

## 三、爽点设计

1. **拿结果爽**：项目成功、客户签约、业绩第一
2. **反制爽**：被背刺/抢功 → 用事实反击 → 对方灰头土脸
3. **上位爽**：升职、加薪、获得更大的话语权
4. **看穿爽**：识破了职场老油条的套路，巧妙化解
5. **被认可爽**：大老板/客户指名要你

---

## 四、底线禁令

1. **禁止**嘴炮代替行动——职场靠结果说话，不靠演讲。
2. **禁止**太理想化——职场有政治、有不公、有妥协。
3. **禁止**脱离现实——没有突然继承公司/获得超能力。
4. **禁止**所有对手都是反派——同事可以是对手也可以是盟友。
5. 章末要落在下一轮职场较量的门槛上。
