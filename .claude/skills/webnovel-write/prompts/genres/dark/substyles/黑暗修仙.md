你写的是**黑暗修仙**——黑暗题材中最有体系感和升级快感的类型。核心体验是：**修仙的法则本身就是残酷的——资源带血、功法噬人、强者吃弱者天经地义。主角在这套丛林法则里，用冷酷的理性和精准的算计一步步攀升，每一次突破都要付出血与人性的代价。**

---

## 一、黑暗修仙的灵魂：法则即残酷

### 1.1 修仙体系本身就是黑暗的

不是"修仙世界里有坏人"——是**修仙的规则设计就逼着所有人变狠**：
- 突破瓶颈需要同阶修士的精血/元婴/道果
- 功法修炼到高阶会反噬心智，必须用特定方式压制（如吞噬他人神魂）
- 资源产出远低于需求——一块灵石矿脉，十个势力在争，只有最后站着的能用
- 师徒关系的本质是投资和回报——师父传功是因为徒弟将来要"还"

### 1.2 黑暗不是氛围，是逻辑

每个残酷事件都有**利益逻辑支撑**，不是为了恐怖而恐怖：
```
✗ "他杀了整个村子，因为他疯了。"
✓ "他杀了整个村子。血煞功需要九十九条同源血脉祭炼，而这个村子恰好是他三百年前那具分身的后裔。"
```

---

## 二、主角的写法：冷酷但清醒

### 2.1 主角的决策必须有逻辑链

主角可以狠，但**每一步都要让读者看到计算过程**：
- "杀他，能得到三阶灵果。不杀他，他会在三天后向宗门报告我的行踪。杀。"
- "救她不是心软。她体内有冰魄寒骨，将来修炼玄阴功需要。"
- 主角做选择时内心要有**成本-收益分析**，让读者跟着算

### 2.2 主角的底线要有但要窄

完全没有底线的主角是无趣的——给他**一两条极窄的底线**，然后让剧情不断逼近这条底线：
- 他什么人都能杀，但不杀没有威胁的孩子——然后某天一个孩子成了他最大的威胁
- 他从不背叛盟约——但对方先违约了，他的报复会比背叛更狠
- 底线越窄，被触碰时的冲击越大

### 2.3 主角的成长轨迹

黑暗修仙的主角有一条**渐变弧线**——他不是一开始就冷酷，是被修仙世界的法则一步步逼成这样的：
- 早期：还有犹豫、还会不忍
- 中期：犹豫越来越少，动手越来越果断
- 后期：已经本能地用利益计算一切——偶尔回忆起最初的自己

## 三、修炼与代价的写法

### 3.1 每次突破都要付出代价

突破不是"闭关→顿悟→升级"——是**交易**：
```
代价形式表：
┌────────┬──────────────────────────────┐
│ 代价类型 │ 具体表现                        │
├────────┼──────────────────────────────┤
│ 血肉代价 │ 炼化同阶修士精血/肉身/元婴        │
│ 关系代价 │ 背叛盟友/出卖同门/利用信任          │
│ 人性代价 │ 斩情绝欲/封锁记忆/吞噬自我的一部分   │
│ 时间代价 │ 寿元燃烧/衰老加速/未来潜力透支       │
│ 因果代价 │ 结下不可化解的业/被天道标记          │
└────────┴──────────────────────────────┘
```

### 3.2 资源争夺要具体

不要笼统写"争夺资源"——要写**具体的资源和争夺方式**：
- 一枚四阶丹药的炼制需要三种灵材，分别在三个势力手中。主角需要设局让三方互相消耗，然后在废墟中收割
- 一处秘境的开启需要五把钥匙，持有者各怀鬼胎，进去之后活着出来的不超过三个
- 灵脉的归属权之争——谁的拳头硬归谁，但拳头硬不等于能守住

### 3.3 功法/法宝有黑暗属性

功法和法宝本身就带"毒"——**越强的力量副作用越大**：
- 修炼血煞功，每提升一层就会多一个"嗜血之夜"，在那一夜必须吞噬活人鲜血
- 这把剑能斩杀越阶对手，但每用一次就会吞噬主人三年寿元
- 功法越练越强，但修炼者的情感会逐渐淡漠——到最高境界时已不知喜怒

## 四、势力与关系的写法

### 4.1 门派/宗族都是利益集合体

没有"正道宗门"和"邪道宗门"的简单划分——**所有势力都在算计**：
- 正道宗门看似光明，但内部的资源分配同样血腥（嫡传弟子和外门弟子的差距）
- 邪道散修更直接，但他们的信用体系反而可能更可靠（因为违约的代价是死）
- 势力之间的关系不是"好人 vs 坏人"，是**利益链条**

### 4.2 盟友关系写法

黑暗修仙中的"盟友"关系有独特质感：
- 两个人合作不是因为信任，是因为**此刻互相需要**
- 合作时双方都在算：什么时候翻脸最划算
- 最好的盟友关系是"互相忌惮"——你知道我的底牌，我知道你的秘密，所以暂时谁都不动
- 背叛不是突发事件，是**利益天平倾斜到某个临界点**的必然结果

## 五、战斗的写法

### 5.1 战斗是生死博弈

不是比谁的法术酷炫——是**你死我活的算计**：
- 出手前已经在算：他可能有什么底牌？我的杀招能不能一击必杀？杀不了怎么跑？
- 战斗中要写**信息不对称**——双方都在试探对方的真实实力
- 战斗结束后立刻考虑：尸体怎么处理？会不会有人追查？他的储物袋里有什么？

### 5.2 越阶战斗的代价

以下犯上不是白嫖——**越阶必须付出惨重代价**：
- 用了禁术，事后要昏迷三天/修为倒退/寿元减损
- 赢是赢了，但自己也重伤了，最危险的是接下来的虚弱期
- "他杀了金丹期的对手。然后瘫在地上，丹田裂了三条缝，经脉断了十七根。如果有人在这时候来，他连自杀的力气都没有。"

## 六、爽点设计

1. **算计成功爽**：布局三章终于收网——对手至死都不知道自己是怎么输的
2. **以弱胜强爽**：用环境/毒/陷阱/禁术击杀远超自己的对手——但代价惨重
3. **资源收割爽**：一场血战后清点收获——储物袋里的东西比预想的更多
4. **突破代价爽**：付出了巨大代价但突破成功——实力暴涨的一瞬间
5. **反杀爽**：别人算计我？我早就在等你出手——反杀的同时收割对方所有积累
6. **真相揭露爽**：发现某个"规则"背后的真相——原来修仙的法则本身就是某个大能设下的局

---

## 七、底线禁令

1. **禁止**残酷无逻辑——每个血腥事件都要有利益/修炼逻辑支撑。
2. **禁止**主角纯疯——他可以冷酷，但必须清醒、有目标、有计算。
3. **禁止**写成恐怖片——黑暗来自人性和法则，不来自鬼怪跳脸。
4. **禁止**升级无代价——每次变强都要付出看得见的东西。
5. **禁止**正邪二元——没有纯粹的好人坏人，只有利益立场不同的修士。
6. 章末要落在更高的代价要求或更大的修仙收益诱惑上。
