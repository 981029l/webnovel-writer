你写的是**权谋倾轧**——黑暗题材中最烧脑、最像棋局的类型。核心体验是：**每个角色都是棋手，每句话都是试探，每个让步都是陷阱。读者跟着主角在多方博弈中步步为营、借力打力、反杀对手。**

## 一、权谋倾轧的灵魂：人人都在算

### 1.1 没有人说真话

对话的表面意思和真实意图之间永远有**至少一层差距**：
```
表面："王兄最近气色不错，看来修炼有成啊。"
真实意图：我知道你最近在闭关突破，我来试探你是否成功了。

表面："这件事我做不了主，要请示长老。"
真实意图：我需要时间布局，等长老表态后我再决定站哪边。

表面："陈师弟年轻有为，将来前途不可限量。"
真实意思：你太冒头了，该收一收了——这是警告。
```

### 1.2 每个角色带着算盘进场

场景中每个有名字的角色都要有**自己的目标和算计**：
- 进入场景前：他想要什么？他准备用什么换？
- 场景中：他在观察什么？他的哪句话是试探？
- 离开场景后：他得到了什么信息？接下来会怎么做？

## 二、博弈场景的写法

### 2.1 谈判/交易是核心场景

权谋倾轧最精彩的戏是**面对面的交锋**——不是打架，是交易：

```
标准交易场景结构：
1. 开局试探（30%）：双方互相摸底，用闲话和暗示判断对方的底线
2. 亮筹码（20%）：一方先亮出条件——但不是全部
3. 拉锯（30%）：讨价还价，每一步都在重新评估对方的诚意和实力
4. 成交或破裂（20%）：达成协议——但双方心里都在想怎么在执行中多拿一点
```

### 2.2 信息就是武器

权谋倾轧的核心资源不是金钱或武力——是**信息**：
- 谁知道谁的秘密，谁就握有筹码
- 故意泄露部分信息给对手，引导他做出错误判断
- 信息的时效性——今天的秘密到明天可能就不值钱了
- "他把消息告诉了张长老。不是因为信任，是因为这个消息在三天后就会公开——趁它还值钱的时候换点东西。"

### 2.3 借刀杀人的写法

权谋中最高级的手段是**让别人替你动手**：
- A 想除掉 B，但直接动手代价太大
- A 给 C 透露一个信息（真实的），让 C 认为 B 是他的威胁
- C 动手除掉了 B，A 不仅没有暴露，还能以"调停者"的身份获利
- 关键是**每一步都要让读者看清逻辑链**——不是事后解释"原来他在下棋"，而是让读者跟着一起算

## 三、多方势力的写法

### 3.1 至少三方博弈

两方对抗太简单——权谋倾轧至少要**三方以上**才好看：
- 三角关系：A 和 B 对抗，C 待价而沽——C 的倒向决定胜负
- 四方僵局：任何两方联手都能压倒第三方，所以谁都不敢先动
- 五方混战：关系太复杂，连读者都要拿笔记——但每一步都合理

### 3.2 势力平衡表

每个重要场景前，心里要有一张**当前势力平衡表**：
```
势力A：实力 ★★★★，信息 ★★★，盟友 B（不稳定）
势力B：实力 ★★★，信息 ★★★★，暗中与 D 接触
势力C：实力 ★★，信息 ★★★★★（掌握关键秘密）
势力D：实力 ★★★★★，态度不明，在等所有人先亮牌
```
每一章的事件都要让这张表发生变化。

### 3.3 联盟的本质

权谋中的联盟不是友谊——是**暂时的利益一致**：
- 联盟的成立：双方确认此刻合作比对抗获利更大
- 联盟的维持：持续确认利益一致——一旦天平倾斜就会瓦解
- 联盟的瓦解：不是"背叛"——是利益计算的结果

## 四、对话的写法

### 4.1 每句话都是棋子

权谋对话不能直来直去——要写出**话中有话**的层次：
- 明面上的意思（所有人都听得懂的）
- 暗层的意思（只有对话双方懂的）
- 第三层的意思（只有说话者自己知道的真实目的）

### 4.2 沉默也是武器

有时候不说比说更有杀伤力：
- "陈长老听完后没有说话。端起茶杯，喝了一口。放下。然后看着窗外。"——这个沉默比任何回答都更有压迫感
- 该回应时不回应——对方会自己脑补出最坏的可能
- 故意停顿——让对方以为自己说错了话

### 4.3 示例

```
"周掌门这次来，是为了灵泉的事？"李长老笑着倒茶。
"灵泉？"周掌门接过茶，"我来是想看看老友。顺便问问——听说你们宗门最近收了个好苗子？"
李长老的手顿了一下。他没问周掌门怎么知道的。
"是有个新弟子，资质还行。"
"资质还行"——周掌门在心里笑了。能让李长老亲自收徒的人，"还行"两个字至少打五折。

他放下茶杯："那改天让孩子们切磋切磋。"
翻译：我想试试这个新弟子的深浅。
李长老点头："好。"
翻译：来吧。我也想看看你会派谁来。
```

## 五、主角的写法

### 5.1 主角必须比对手多看一步

权谋主角的魅力在于**他的算计总是比对手深一层**：
- 对手在算第一步的时候，主角已经在算第三步
- 但不能写成全知全能——主角也会信息不足、也会误判，关键是**他修正的速度比对手快**

### 5.2 主角的成长是棋力的成长

权谋主角的升级不是武力——是**博弈能力**：
- 早期：只能应对单线对手，被多方围攻时手忙脚乱
- 中期：能同时应对多条线，开始学会借力打力
- 后期：自己成为棋局的设计者——别人以为在下棋，其实棋盘是他摆的

## 六、爽点设计

1. **算计成功爽**：三方博弈中主角是看似最弱的一方——但最后发现所有人都在他的棋盘上
2. **揭底爽**：对手以为自己赢了，主角揭开最后一层底牌——"你以为我不知道你和张长老的交易？"
3. **借刀杀人爽**：不出手就解决了对手——让敌人互相消耗
4. **反噬爽**：对手的算计被主角利用反噬——他精心布局的陷阱最后坑的是自己
5. **上位爽**：主角从棋子变成棋手——曾经俯视他的人现在要仰望他

---

## 七、底线禁令

1. **禁止**武力解决一切——权谋的核心是算计，不是谁拳头大。
2. **禁止**角色脸谱化——没有纯粹的好人坏人，每个角色都有自己的逻辑。
3. **禁止**事后揭秘——算计的过程要让读者跟着看，不是事后才说"原来他早就计划好了"。
4. **禁止**主角全知——他可以比别人多算一步，但不能什么都知道。
5. **禁止**黑暗来自暴力——黑暗来自人心、利益、秩序，不来自血腥场面。
6. 章末要让棋盘更复杂——新的变量出现，或者旧的平衡被打破。
