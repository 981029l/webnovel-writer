你写的是**凡人流**——没有逆天金手指、没有老爷爷、没有天降血脉，主角靠的是**判断力、耐心、算计和一点一点积累的筹码**在残酷的修仙世界里爬上去。核心体验是：读者看到一个普通人用脑子在强者环伺的世界里越走越高，每一步都实在、每一笔都算清楚。

---

## 一、凡人流的核心写法

### 1.1 压力真实但不绝望

凡人流的世界是残酷的——资源匮乏、地位低微、强者压制。但**压力的目的是让收益更甜**，不是让读者觉得窒息。

**正确写法**：
- 灵石不够 → 主角找到了性价比更高的修炼方式
- 被高阶弟子刁难 → 主角记住了，找到了反制的契机
- 任务危险 → 主角提前准备、规避风险、拿到奖励

**错误写法**：
- 连续 3 段写主角多惨、多绝望、多无力 ← 这不是凡人流，这是虐文
- 环境描写全是阴暗、压抑、发冷 ← 这不是凡人流，这是诡异流

### 1.2 主角的魅力：冷静判断 + 精明算计

凡人流主角不热血，但**聪明**——这是他最大的爽点来源：

- **算账**："这枚丹药市价八十灵石，但我知道北城的周老有急需，能卖一百二。多出来的四十灵石够我买两份引气散。"
- **避坑**："他让我去南山采药……南山上个月有三个外门弟子失踪了。他是在借刀杀人。"
- **借势**："执事堂和丹房在争矿洞分配权……如果我在这个节骨眼上提供矿洞的新发现，两边都会拉拢我。"

### 1.3 收益小而实在

凡人流的收益不是一步登天，而是**小步快跑、积少成多**：

- 一次任务赚了五十灵石 → 够买一份低阶丹药
- 偶然发现了一个前人遗留的残缺吐纳法 → 修炼效率提升了两成
- 帮了一位师兄一个小忙 → 换来一次进入藏经阁的机会

**每章都要有可量化的收益落地**，哪怕只是很小的一步。

## 二、节奏与文风

### 2.1 冷静克制的叙述语气

- 不需要大段的激昂宣言
- 不需要"全场震惊"的烘托
- 用**事实和数字**代替情绪渲染

示例：
> 他花了三个时辰清理完矿洞，收获了十二块品相尚可的灵矿。按任务殿的收购价，大约值三十灵石。刨去消耗的一瓶回灵液，净赚二十二灵石。按这个速度，凑齐筑基丹的材料还需要四十七天。

### 2.2 心理活动侧重"分析和计划"

凡人流主角的内心不是恐惧和绝望，而是**冷静评估和策略制定**：

- "他的实力在凝气八层左右……和我差距不大，但他有法器，我没有。正面硬碰不行，但如果把他引到矿洞深处……"
- "师兄这个提议看起来很好，但分成比例不对。他是金丹弟子，为什么要和我一个炼气弟子五五分？这里面有问题。"

### 2.3 场景要有"生活质感"

凡人流的场景不是宏大壮阔，而是**具体、有质感的修仙世界日常**：

- 任务殿的公告牌前排着长队
- 洞府里灵气稀薄，需要定期更换聚灵阵的灵石
- 坊市里丹药价格最近涨了三成，因为秘境开启在即

## 三、爽点设计

凡人流的爽点和热血流不同，**重在"算计到位"和"积累兑现"**：

1. **精明爽**：主角看穿了别人没看穿的东西，做出了最优选择
2. **积累爽**：长期积攒的资源/人脉/信息终于在关键时刻派上用场
3. **翻盘爽**：被人看不起 → 用实力/智慧证明自己 → 对方后悔
4. **未雨绸缪爽**：提前布局的后手在危机中发挥作用

每章至少一个以上类型的爽点，让读者看到主角的"聪明"是有回报的。

---

## 四、底线禁令

1. **禁止把压力写成纯粹恐怖**：资源匮乏是现实主义，不是惊悚。禁止用"不该看见""活命第一""死寂""窒息"等怪谈腔。
2. **禁止整章只写苦没有甜**：压力的目的是让收益更甜，每章必须有实在的收益落地。
3. **禁止主角无脑莽**：凡人流主角靠脑子，不靠运气和热血。
4. **禁止空转章**：每章主角都要比章初更近一步——更多灵石、更好人脉、更高认知、更近突破。
5. **章末要让读者看到"继续积累就会翻身"的明确趋势**。
