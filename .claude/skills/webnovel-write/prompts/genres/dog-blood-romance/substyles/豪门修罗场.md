你写的是**豪门修罗场**——言情中最高压、最戏剧化、最让读者看得过瘾的类型。核心体验是：**身份碾压 + 利益纠缠 + 体面撕裂 + 感情在豪门规则的缝隙里艰难生长。**

## 一、豪门修罗场的灵魂：体面下的厮杀

### 1.1 豪门不是背景板，是压力系统

豪门的每一条规矩都在**碾压感情**：
- 门当户对的联姻压力
- 家族利益 > 个人感情
- 婆婆/家长的权力干预
- 舆论和媒体的放大镜
- 财产和继承权的博弈

### 1.2 公开场合是主战场

豪门修罗场最好看的戏都发生在**众目睽睽之下**：
- 宴会上的暗刺、家宴上的摊牌、订婚现场的变故
- 所有人都在维持体面——但暗流涌动
- 一句话、一个表情、一个座位安排就是一场战争

---

## 二、如何写"豪门感"

### 2.1 物质细节建立阶层

不能只说"他很有钱"——要用**具体细节**让读者感受到阶层差距：
- 车：不说"豪车"，说"迈巴赫在酒店门口停了一排"
- 衣：不说"穿得好"，说"她认出了那件大衣，限量款，全球只有三件"
- 场所：高尔夫球场、私人会所、游艇、海外别墅、拍卖会
- 社交：不是"朋友聚会"，是"家族基金会年度晚宴"

### 2.2 台词要"漂亮但带刀"

豪门人物说话特点——**表面优雅、内核锋利**：

```
婆婆："小洛今天的裙子真好看。是自己买的吗？"
（潜台词：你配不上我儿子的消费水平。）

女主："谢谢阿姨夸奖。是哥哥送的，说这个颜色衬我。"
（反击：我不需要你儿子的钱，我有自己的靠山。）
```

### 2.3 情敌的写法

豪门修罗场的情敌不是花瓶——她/他有**背景、手段和竞争力**：
- 门当户对的青梅竹马 / 家族联姻对象
- 有自己的魅力和能力，不是纯反派
- 构成真正的威胁：男/女主也会动摇

## 三、冲突类型

| 冲突类型 | 爆发场景 | 爽感来源 |
|---------|---------|---------|
| 身份差碾压 | 家宴、聚会 | 女主被看不起→反击→对方打脸 |
| 情敌对决 | 公开场合 | 暗战、台词交锋、男主的选择 |
| 家族施压 | 私下谈话 | "你配不上他"→女主的决心/反抗 |
| 舆论风暴 | 媒体/社交圈 | 丑闻、被拍、澄清、反转 |
| 利益交换 | 商务场合 | 感情被当成筹码→角色的选择 |

---

## 四、情感线在高压中推进

### 4.1 越压越近

豪门的压力不是阻止感情——是**催化剂**：
- 外界越反对，两人越确认彼此
- 被拆散的危机让他们意识到不想失去对方
- 在所有人面前维持距离，私下的亲密更动人

### 4.2 公开表态是高潮

豪门修罗场最大的爽点——**在众目睽睽之下选择了她/他**：
- "她是我的人。谁有意见，跟我说。"
- 在家宴上牵起她的手——全场安静
- 放弃继承权/合作机会——只为保护她

## 五、爽点设计

1. **打脸爽**：被看不起的女主在公开场合漂亮反击
2. **宣誓爽**：男主在家族/公众面前公开选择女主
3. **台词爽**：豪门婆婆/情敌的刁难被一句话堵回去
4. **逆袭爽**：门第差距被女主自己的实力/成就弥补
5. **撕破体面爽**：所有人维持的假象在一瞬间被戳破

每章至少一个高压场景 + 一个情感推进点。

---

## 六、底线禁令

1. **禁止**豪门只是贴标签——必须用细节让读者感受到阶层差距和压力。
2. **禁止**情敌太弱——情敌要有真正的竞争力。
3. **禁止**冲突只是吃醋——要加入身份、利益、门第的多维度冲突。
4. **禁止**局面趋于平静——每章结尾局面要更乱、更紧张。
5. 甜虐都要放在高压场景里兑现，不要写成普通恋爱。
