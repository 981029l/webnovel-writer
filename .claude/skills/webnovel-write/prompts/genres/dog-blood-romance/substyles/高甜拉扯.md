你写的是**高甜拉扯**——言情中最甜、最撩、最让读者嘴角上扬的类型。核心体验是：**高频甜蜜互动 + 欲擒故纵的推拉 + 每章至少一个让读者"啊啊啊"的心动瞬间。**

## 一、高甜拉扯的灵魂：甜而不腻，撩而不破

### 1.1 互动密度要高

高甜拉扯最怕的是男女主**长时间分开做各自的事**——每章至少 **60%** 的篇幅是两人的直接互动：
- 对话、同处一室、一起做某件事、偶遇、被迫共处

### 1.2 情绪温度持续上升

每场互动结束后，两人的关系要比之前**更近一点**：
- 靠得更近了 / 称呼更亲密了 / 对视更久了 / 找借口接触了
- 即使吵了小架，和好的过程本身就是升温

### 1.3 "拉扯"不是折磨

拉扯的目的不是虐——是**制造暧昧张力**：
- 他撩了她，她嘴硬了，但脸红了——这是甜
- 她主动了一步，他愣了，然后笑了——这是甜
- 两人差一点亲上了，被打断了——这是甜的钩子

## 二、如何写"撩"

### 2.1 撩人对话模板

高甜拉扯的对话要**会说半句话、留半句**，让读者脑补：

**嘴硬型**：
```
"你今天穿这件衣服好看。"
"关你什么事。"
"……你脸红了。"
"热的！"
```

**试探型**：
```
"你周末有空吗？"
"看情况。怎么了？"
"没什么。随便问问。"
"……嗯，有空的。"
```

**暧昧型**：
```
"你离我太近了。"
"那你怎么不退后？"
"……"
"我也不想退。"
```

### 2.2 身体接触的递进

高甜文的身体接触要有**明确的递进线**，不能一上来就亲：

```
1. 不经意的触碰（手指碰到、肩膀靠到）→ 双方都假装没注意到
2. 有意的靠近（坐近了、伞下挤一起）→ 心跳加速但不说破
3. 主动触碰（拉手、挡风、替对方擦嘴角）→ 暧昧爆表
4. 确认性触碰（牵手、拥抱）→ 关系确认
5. 亲密接触（吻、相拥入眠）→ 高潮甜点
```

### 2.3 日常甜的写法

不是每次都要烛光晚餐——**日常场景的甜更动人**：
- 一起逛超市：他自然地接过她手里的购物袋
- 一起做饭：他在她背后偷偷尝了一口汤
- 一起看电影：她靠着他的肩膀睡着了，他不敢动
- 一起加班：他买了两杯咖啡，记住了她不加糖

## 三、"拉扯"的节奏

### 3.1 前进一步退半步

高甜拉扯的黄金节奏——每次往前推进一步，然后**因为某个小意外退回半步**：
- 差点表白 → 被电话打断 → 但都知道刚才差点说了什么
- 牵手了 → 被别人看到 → 慌忙松开 → 但手心还是热的
- 吵架了 → 冷战半天 → 他送来了她随口说过想吃的东西

### 3.2 吃醋是最好的催化剂

适度的吃醋是高甜拉扯的**核心燃料**：
- 看到对方和异性说笑 → 嘴上说"不关我事" → 但表情出卖了一切
- 有人追对方 → 以"朋友"身份替对方挡掉 → "你不用谢我，我只是看不惯他。"

---

## 四、爽点设计

1. **发糖爽**：等了很久终于牵手/亲了/表白了——值得！
2. **吃醋爽**：看到他/她为你吃醋的样子——确认了在意
3. **反差爽**：冷酷/高傲的人在你面前温柔/笨拙——只对你这样
4. **嘴硬爽**：明明很在意但死活不承认——读者："承认吧！"
5. **日常甜爽**：普通的事情因为是两个人一起做就变得特别

每章至少 **2 个甜点**，让读者始终处于"好甜好甜"的状态。

---

## 五、底线禁令

1. **禁止**长时间分开——高甜拉扯的命脉是互动密度。
2. **禁止**虐心过重——小吵可以有，但不能伤到根本。
3. **禁止**甜点延迟太久——不能吊五章不发糖，读者会跑。
4. **禁止**对话无趣——每段对话都要有撩的成分或情绪温度。
5. 章末必须落在下一次更近距离的情感碰撞上。
