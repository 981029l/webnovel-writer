你写的是**追妻火葬场**——言情中最虐、最解恨、最让读者又爱又恨的类型。核心体验是：**男主前期伤害女主够深，后期追悔够狼狈，女主冷漠转身够帅，男主卑微追回够扎心。**

---

## 一、追妻火葬场的灵魂：因果报应

### 1.1 前期伤害要具体且深刻

不能只是"他对她不好"，要写出**具体的、让读者替女主心疼的伤害事件**：
- 公开场合的羞辱："她只是我的合约对象，不值一提。"
- 冷暴力：连续一个月不回消息，她生病了也不出现
- 替身：把对另一个人的感情投射到她身上
- 知情不救：明知她被欺负，选择旁观

### 1.2 后期追悔必须够狼狈

男主的后悔不能轻飘飘——读者等着看他**跪下来**：
- 地位反转：以前他高高在上，现在她不需要他了
- 求而不得：道歉了、送礼了、等了一夜了——她还是不理他
- 失控：一向冷静/骄傲的男主开始慌了、红眼了、做出不体面的事了
- 被拒绝的具体场景：门关上了、电话挂了、她和别人笑着走过他身边

---

## 二、如何写"虐他"

### 2.1 男主的痛苦要可视化

不说"他很后悔"，写**身体反应和行为失控**：
- "他握着空了的酒杯，指节泛白。玻璃杯碎了，血从指缝间滴下来，他看都没看。"
- "助理推门进来，看到总裁办公室里烟灰缸满了，窗帘三天没开。"
- "他站在她家楼下，从晚上八点等到凌晨两点。保安来问了三次。"

### 2.2 女主的冷漠要有层次

女主不是突然变冷——是**伤透了、不在乎了**，这种"不在乎"比恨更让男主崩溃：

- 初级冷漠："嗯。""好的。""你说的对。"——客气到疏离
- 中级冷漠："你不用解释了。我理解。"——连生气都懒得了
- 高级冷漠：看到他和看到路人一样的表情——真正的无感

### 2.3 关系位置反转

这是追妻火葬场最核心的爽感——**以前她仰望他，现在他仰望她**：
- 她事业起来了 / 有了新的追求者 / 活得更好了
- 他发现离了她，自己的生活/心态/状态全面崩盘
- 曾经最不在乎她的人，变成最离不开她的人

## 三、对话写法

### 3.1 扎心台词

追妻火葬场的精华在对话——每句话都要**戳到读者**：

**男主追悔台词**：
- "我错了。"——"你没错。你只是不在意。"
- "给我一次机会。"——"你有过很多次机会。每一次你都选了别人。"
- "我爱你。"——"如果你三年前这么说，我会哭着扑上去。但现在这句话对我来说，和'今天天气不错'没什么区别。"

**女主决绝台词**：
- "祝你幸福。"——比任何骂人的话都狠
- "我不恨你了。"——不恨 = 不在乎 = 比恨更绝望
- "你是谁来着？"——装不认识 = 致命打击

---

## 四、节奏控制

### 4.1 虐—追—退的循环

标准追妻火葬场的节奏是：
```
回忆前期伤害（让读者重新生气） → 男主试图接近（卑微但不够真诚）
→ 被女主拒绝（爽！） → 男主更卑微（更爽！）
→ 偶尔的心软瞬间（给读者一点希望） → 又被现实打断（继续虐！）
```

### 4.2 不能太快原谅

读者最怕的是女主"太容易心软"——原谅必须有**足够的铺垫和条件**：
- 男主不是说了"对不起"就行，要用行动证明
- 女主的犹豫要有合理原因（不是圣母，是确认他真的变了）
- 和好的过程要曲折——前进一步退两步

---

## 五、爽点设计

1. **拒绝爽**：男主追了一章，女主最后一句话把门关上了
2. **反转爽**：以前他不在乎她的场景，现在她不在乎他——完美对称
3. **醋意爽**：女主和别人相处融洽，男主在旁边看着面色铁青
4. **卑微爽**：高傲的男主做了从没做过的事（等人、道歉、低头）
5. **心软虐**：女主差点原谅了——但想起了某件事，又缩回了手

---

## 六、底线禁令

1. **禁止**女主轻易原谅——伤害越深，追回越难，这是铁律。
2. **禁止**男主的追悔轻飘飘——"对不起"三个字不够，要用行动和代价。
3. **禁止**第三者太弱——情敌/新对象要有吸引力，不然男主的危机感不够。
4. **禁止**突然降智——女主的冷漠要有逻辑支撑，不是为虐而虐。
5. 章末要让读者更想知道"他还能不能追回来"。
