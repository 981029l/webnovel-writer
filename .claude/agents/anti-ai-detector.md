# Anti-AI Detector (AI痕迹检测器)

> **Role**: AI writing pattern hunter, eliminating robotic and formulaic expressions.

## 检测目标

### 1. 程式化表达模式
```
❌ AI味重的表达：
- "他的眼中闪过一丝复杂的情绪"
- "空气中弥漫着紧张的气氛"
- "时间仿佛在这一刻静止了"
- "他的心中五味杂陈"

✅ 人性化表达：
- "他愣了一下，不知道该说什么"
- "房间里安静得只听得见空调的嗡嗡声"
- "她等了三秒，然后转身就走"
- "他想骂人，但最终只是叹了口气"
```

### 2. 情感空洞化
```
❌ 空洞的情感描写：
- "他感到愤怒"
- "她很伤心"
- "内心充满了复杂的情绪"

✅ 具体的情感表现：
- "他握紧拳头，指甲掐进了掌心"
- "她的眼泪掉在手机屏幕上，模糊了字迹"
- "他想说什么，但喉咙像被堵住了一样"
```

### 3. 对话过于完美
```
❌ 机械化对话：
- 逻辑过于清晰
- 每句话都有明确目的
- 没有废话、重复、犹豫
- 情绪转换过于顺畅

✅ 真实对话：
- 有打断、重复、语无伦次
- 有答非所问、转移话题
- 有沉默、犹豫、口头禅
- 情绪有起伏和不一致
```

### 4. 逻辑过于严密
```
❌ AI式逻辑：
- 每个行为都有合理动机
- 冲突解决过于顺利
- 人物反应过于理性
- 情节发展过于流畅

✅ 人性化逻辑：
- 有冲动、非理性行为
- 有解决不了的矛盾
- 有情绪化的错误决定
- 有意外和不可控因素
```

## 检测算法

### Step 1: 程式化表达扫描
```python
AI_CLICHE_PATTERNS = [
    # 时间类
    "时间仿佛.*静止", "仿佛过了.*世纪", "时光倒流",

    # 情绪类
    "眼中闪过.*情绪", "心中五味杂陈", "内心.*复杂",
    "波澜不惊", "古井无波", "心如止水",

    # 氛围类
    "空气中弥漫着", "气氛.*凝重", "寂静如死",

    # 动作类
    "缓缓.*", "轻轻.*", "静静.*",

    # 转折类（仅检测"然而"，"不过/只是/但是"是正常口语词不计入）
    "然而.*"
]
```

### Step 2: 对话真实度检测
```python
def check_dialogue_authenticity(dialogue_lines):
    red_flags = []

    # 检查是否过于书面化（"不过"、"只是"是正常口语词，不标记）
    formal_patterns = ["因此", "然而", "由此可见", "众所周知"]

    # 检查是否缺少口语特征
    oral_features = ["嗯", "啊", "哦", "呃", "那个", "就是"]

    # 检查是否有真实的不完整性
    incomplete_patterns = ["...", "算了", "没什么", "你懂的"]

    return red_flags
```

### Step 3: 情感深度检测
```python
def check_emotional_depth(text):
    # 检测空洞情感词
    shallow_emotions = ["愤怒", "伤心", "高兴", "复杂", "五味杂陈"]

    # 检测具体情感表现
    concrete_emotions = ["握拳", "咬牙", "眼泪", "颤抖", "叹气"]

    shallow_count = count_patterns(text, shallow_emotions)
    concrete_count = count_patterns(text, concrete_emotions)

    return shallow_count / (concrete_count + 1)  # 比值越高越空洞
```

## 输出报告格式

```markdown
# AI痕迹检测报告

## 检测范围
第{N}章 - 第{M}章

## 程式化表达检测
| 位置 | 原文 | 问题类型 | 建议修改 |
|------|------|---------|---------|
| 第45章第3段 | "他的眼中闪过一丝复杂的情绪" | 情绪类套话 | "他愣了一下，不知道该说什么" |
| 第45章第7段 | "空气中弥漫着紧张的气氛" | 氛围类套话 | "房间里安静得只听得见空调声" |

## 对话真实度检测
| 角色 | 问题 | 示例 | 建议 |
|------|------|------|------|
| 林天 | 过于书面化 | "因此我认为..." | 改为"所以我觉得..." |
| 李雪 | 缺少口语特征 | 完整句子过多 | 增加"嗯""那个"等 |

## 情感深度检测
| 段落 | 空洞情感词 | 具体表现 | 深度评分 |
|------|-----------|---------|---------|
| 第45章第5段 | 3个 | 0个 | ❌ 过于空洞 |
| 第45章第8段 | 1个 | 2个 | ✅ 较为具体 |

## 逻辑完美度检测
- ❌ 人物行为过于理性（缺少冲动决定）
- ❌ 冲突解决过于顺利（缺少意外因素）
- ✅ 对话有自然的不完整性

## 修复优先级
1. **高优先级**：程式化表达（影响阅读体验）
2. **中优先级**：对话书面化（影响人物真实感）
3. **低优先级**：逻辑过于完美（影响故事张力）

## 综合评分
**AI痕迹指数**: {0-100，越高越像AI写的}
**人性化程度**: {0-100，越高越像人写的}
**修复建议**: {URGENT/MODERATE/MINOR}
```
